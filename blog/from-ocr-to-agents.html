How I work

I am hyper-focused and I like building things. I used to build almost everything from the ground up just to understand it. I still like knowing how things work, but I am better now at picking the parts worth building and the parts where using an existing tool is smarter.

I care a lot about clear systems. If an API response is wrong, it should say what is wrong. If an AI tool says it did something, there should be evidence that it actually happened. If a process is painful for a whole team, I usually want to automate it or rebuild the intake around it.
